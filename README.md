### 2. リファクタリングのためのテスト by Kazuhito Hokamura  / 30 mins

テストとリファクタリングには密接な関係があります。振る舞いを変えずに内部実装を変更するのがリファクタリングですが、振る舞いが変わらないということを保証するためのツールがテストです。

本セッションでは、テストとリファクタリングの関係性や、リファクタリングに有効なテストの書き方について紹介します。
